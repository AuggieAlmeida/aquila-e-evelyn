# Aquila & Evelyn - Convite de Casamento

Um site elegante e responsivo para o convite de casamento de Aquila e Evelyn.

## Sobre

Este projeto é um convite digital para o casamento, com design sofisticado e moderno. O site apresenta uma página inicial e uma página de convite com detalhes do evento, data, horário e local.

**Data do Casamento:** Domingo, 03 de Maio de 2026  
**Horário:** 16h30  
**Local:** Rua Oratório, 3921 - Santo André/SP

## Características

- Design responsivo (mobile, tablet e desktop)
- Paleta de cores elegante (verde sage, lavanda, ouro)
- Animações suaves e fluidas
- Página de convite com efeito de carta
- Botão de selo de cera interativo
- Elementos decorativos flutuantes
- Tipografia refinada com múltiplas fontes

## Estrutura do Projeto

```
casamento/
├── index.html                  # Página principal
├── README.md                   # Este arquivo
├── assets/                     # Recursos de mídia
│   ├── gallery/               # Galeria de fotos
│   ├── icons/                 # Ícones (calendário, relógio, localização)
│   ├── flower.png             # Imagem de flor
│   └── leaf.png, leaf2.png    # Imagens de folhas
├── css/
│   └── styles.css             # Estilos da página principal
├── js/
│   └── script.js              # Scripts da página principal
└── invite/
    ├── invite-page.html       # Página de convite
    ├── invitation-styles.css  # Estilos da página de convite
    └── invitation-script.js   # Scripts da página de convite
```

## Páginas

### 1. Página Principal (`index.html`)
- Bem-vindo ao site do casamento
- Links para acessar o convite digital

### 2. Página de Convite (`invite-page.html`)
- Banner superior (50vh) com apresentação dos noivos
- Seção de carta com detalhes do evento
- Ícones de calendário, relógio e localização
- Botão de selo de cera interativo

## Como Usar

1. Abra o arquivo `invite-page.html` em um navegador web
2. Clique no botão de convite para ir à página inicial (`index.html`)
3. Explore as seções da carta de convite

## Paleta de Cores

- **Verde Sage:** `#5a7f5f` - Cor primária
- **Lavanda:** `#c6a7fe` - Cor secundária
- **Ouro:** `#d4af37` - Cor de destaque
- **Blush:** `#f5e6e8` - Tons suaves
- **Creme:** Fundo geral

## Tipografia

- **Montserrat:** Corpo do texto (sans-serif)
- **Playfair Display:** Títulos (serif)
- **Great Vibes:** Assinaturas (cursive)

## Animações

- Fade In Up: Elementos aparecendo gradualmente
- Float: Movimento suave de elementos decorativos
- Pulse: Efeito de pulso no selo
- Shine: Brilho refletido no selo de cera
- Paper Fold: Animação de desdobramento da carta

## Responsividade

O site é totalmente responsivo com breakpoints em:
- Desktop: 1024px+
- Tablet: 768px - 1023px
- Mobile: < 480px

## Tecnologias

- **HTML5:** Estrutura
- **CSS3:** Estilos e animações
- **JavaScript:** Interatividade
- **Google Fonts:** Tipografia

## Notas

- As imagens dos ícones devem estar em `assets/icons/`
- As imagens decorativas (flores e folhas) devem estar em `assets/`
- O site utiliza `position: relative` e `overflow: hidden` para evitar transbordamento de elementos animados
